# WP Book (wp-book) Plugin
Book Plugin 

## Overview
  1. This plugin will create a Custom post type "Book".
  
  2. Create a custom hierarchical taxonomy Book Category
  
  3. Create a custom non-hierarchical taxonomy Book Tag
  
  4. Create a custom meta box to save book meta information.
  
  5. Create custom meta table and save all book meta information in that table.
  
  6. Create a custom admin settings page for Book.
  
  7. Create a shortcode [book] to display the books information.
  
  8. Create a custom widget.
  
  9. Create a custom dashboard widget.
